# Debating Society, IIEST Shibpur - Official Website

This repository contains the source code for the official static website of the **Debating Society (DebSoc), IIEST Shibpur**. The website serves as a digital hub for information regarding the society's history, upcoming events, and debating resources for the student community.

---

## Project Overview

The website is designed to be a lightweight, fast-loading platform that showcases the society's activities. As a static site, it focuses on clean presentation and ease of maintenance.

## Tech Stack

The project is built using fundamental web technologies to ensure compatibility and performance:

* **HTML5:** Used for structuring the website's content.
* **CSS3:** Used for styling, layout, and responsive design to ensure the site works across all devices.
* **JavaScript (Vanilla):** Used for DOM manipulation, interactive elements, and navigation logic.

---

## Key Features

* **Responsive Design:** Optimized for desktops, tablets, and mobile devices.
* **Event Gallery:** A section dedicated to past tournaments and debating sessions.
* **Member Directory:** Information about the current executive body and active members.
* **Resources Section:** Links and documents to help students learn different debating formats.
